# jsapp-boilerplate

A personal JavaScript boilerplate for frontend applications.

This boilerplate contains:

|              |                                         |
|--------------|-----------------------------------------|
| Presentation | React, CSS modules*                     |
| Store        | Redux*                                  |
| Test         | Karma, Enzyme, Chai, ESLint             |
| Build        | Webpack, Babel, Yarn, TypeScript/Flow?* |
| Deploy       | Docker?*                                |
